================================================================================
BUG FIXES DURING IMPLEMENTATION
================================================================================

1. String Default Constructor Bug:
   - Issue: capacity_ was 0 but allocated 1 byte
   - Fix: Set capacity_ = 1 in default constructor
   
2. String ensure_capacity Bug:
   - Issue: Didn't allocate for empty strings
   - Fix: Always allocate at least 1 byte for null terminator
   
3. HashMap operator[] Bug:
   - Issue: Pointer invalidation after rehash
   - Fix: Re-fetch pointer after insert
   
4. File Double-Free Bug:
   - Issue: Default copy constructor caused double-free
   - Fix: Added move semantics, deleted copy operations
   
5. Math Test Bug:
   - Issue: log(E) should be 1, not 0
   - Fix: Updated assertion to check log(E) ≈ 1.0
