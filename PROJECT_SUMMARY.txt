EXAMPLE CODE
================================================================================

Sapphire Code:
```sapphire
let x = 10
let y = 3.14
let result = x + y  # Type promotion: int + float → float

if result > 10:
    let message = "Greater"
else:
    let message = "Less"

let i = 0
while i < 5:
    let temp = i * 2
    i = i + 1
```

Generated LLVM IR:
```llvm
define i32 @main() {
entry:
  %x = alloca i64
  store i64 10, ptr %x
  %y = alloca double
  store double 3.140000e+00, ptr %y
  %x1 = load i64, ptr %x
  %y2 = load double, ptr %y
  %promoteL = sitofp i64 %x1 to double
  %addtmp = fadd double %promoteL, %y2
  %result = alloca double
  store double %addtmp, ptr %result
  ; ... (control flow continues)
  ret i32 0
}
```

TECHNICAL ARCHITECTURE
================================================================================

Compilation Pipeline:
Source (.spp) → Lexer → Parser → Type Checker → LLVM IR → Native Code

Components:
- Lexer: Tokenization with indentation handling
- Parser: AST generation with error recovery
- Type System: Hindley-Milner inference + generics + traits
- Code Generator: LLVM IR generation
- Optimizer: LLVM optimization passes (coming)
- Linker: Native executable generation (coming)

Technology Stack:
- Language: C++20
- Compiler Infrastructure: LLVM 21
- Build System: Make + CMake
- Platform: Linux (Fedora 43)
