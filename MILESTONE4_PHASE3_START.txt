WHY MEMORY SAFETY MATTERS
================================================================================

Common Memory Errors in C/C++:

1. Use-After-Free (70% of security vulnerabilities!)
```c
char* ptr = malloc(100);
free(ptr);
strcpy(ptr, "hello");  // CRASH or exploit!
```

2. Double-Free
```c
free(ptr);
free(ptr);  // CRASH!
```

3. Buffer Overflow
```c
char buf[10];
strcpy(buf, "this is way too long");  // OVERFLOW!
```

4. Null Pointer Dereference
```c
char* ptr = NULL;
*ptr = 'x';  // CRASH!
```

These bugs cause:
❌ Crashes
❌ Security vulnerabilities
❌ Data corruption
❌ Unpredictable behavior

Sapphire's Solution:
✅ Detect at runtime
✅ Clear error messages
✅ Prevent exploitation
✅ Safe by default

TECHNICAL DESIGN
================================================================================

1. Use-After-Free Prevention
```cpp
// Poison freed memory
void free(void* ptr) {
    ObjectHeader* header = get_header(ptr);
    header->set_freed(true);  // Mark as freed
    memset(ptr, 0xDD, header->size);  // Poison
    actual_free(ptr);
}

// Check on access
void* access(void* ptr) {
    ObjectHeader* header = get_header(ptr);
    if (header->is_freed()) {
        throw UseAfterFreeError(ptr);
    }
    return ptr;
}
```

2. Double-Free Prevention
```cpp
void free(void* ptr) {
    ObjectHeader* header = get_header(ptr);
    if (header->is_freed()) {
        throw DoubleFreeError(ptr);
    }
    header->set_freed(true);
    actual_free(ptr);
}
```

3. Buffer Overflow Detection
```cpp
// Add bounds to arrays
struct Array {
    size_t length;
    void* data;
};

void* array_access(Array* arr, size_t index) {
    if (index >= arr->length) {
        throw BufferOverflowError(index, arr->length);
    }
    return arr->data + index * element_size;
}
```

4. Type Safety
```cpp
void* cast(void* ptr, TypeID expected_type) {
    ObjectHeader* header = get_header(ptr);
    if (header->type_id != expected_type) {
        throw TypeMismatchError(header->type_id, expected_type);
    }
    return ptr;
}
```

5. Null Pointer Checks
```cpp
void* check_null(void* ptr) {
    if (ptr == nullptr) {
        throw NullPointerError();
    }
    return ptr;
}
```

================================================================================
OBJECT HEADER UPDATES
================================================================================

Add new flags to ObjectHeader:

```cpp
struct ObjectHeader {
    uint64_t type_id;
    uint32_t size;
    uint32_t gc_flags;
    
    // New safety flags
    static constexpr uint32_t FREED_BIT = 0x08;
    static constexpr uint32_t POISONED_BIT = 0x10;
    
    bool is_freed() const { return gc_flags & FREED_BIT; }
    void set_freed(bool freed) {
        if (freed) gc_flags |= FREED_BIT;
        else gc_flags &= ~FREED_BIT;
    }
    
    bool is_poisoned() const { return gc_flags & POISONED_BIT; }
    void set_poisoned(bool poisoned) {
        if (poisoned) gc_flags |= POISONED_BIT;
        else gc_flags &= ~POISONED_BIT;
    }
};
```

================================================================================
ERROR HANDLING
================================================================================

Safety Errors:

```cpp
class SafetyError : public std::runtime_error {
public:
    SafetyError(const std::string& msg) : std::runtime_error(msg) {}
};

class UseAfterFreeError : public SafetyError {
public:
    UseAfterFreeError(void* ptr)
        : SafetyError("Use-after-free: " + ptr_to_string(ptr)) {}
};

class DoubleFreeError : public SafetyError {
public:
    DoubleFreeError(void* ptr)
        : SafetyError("Double-free: " + ptr_to_string(ptr)) {}
};

class BufferOverflowError : public SafetyError {
public:
    BufferOverflowError(size_t index, size_t length)
        : SafetyError("Buffer overflow: index " + std::to_string(index) +
                     " >= length " + std::to_string(length)) {}
};

class TypeMismatchError : public SafetyError {
public:
    TypeMismatchError(uint64_t actual, uint64_t expected)
        : SafetyError("Type mismatch: expected " + std::to_string(expected) +
                     ", got " + std::to_string(actual)) {}
};

class NullPointerError : public SafetyError {
public:
    NullPointerError() : SafetyError("Null pointer dereference") {}
};
```

```llvm
%element_ptr = getelementptr ptr %array, i64 %index
%value = load i64, ptr %element_ptr
```

================================================================================
PERFORMANCE CONSIDERATIONS
================================================================================

Safety checks have overhead:
- Null checks: ~1-2 cycles
- Bounds checks: ~5-10 cycles
- Type checks: ~5-10 cycles
- Use-after-free checks: ~10-20 cycles

Mitigation strategies:
1. Optimize hot paths
2. Eliminate redundant checks
3. Use compiler optimizations
4. Make checks optional (release mode)

Expected overhead: 5-10% in debug mode, 0-2% in release mode
