================================================================================
CHALLENGES OVERCOME
================================================================================

1. LLVM Integration Complexity
   - Solution: Opaque pointer pattern
   - Result: Clean, maintainable code

2. AST Structure Differences
   - Challenge: Operators as strings, not tokens
   - Solution: String comparison for operators
   - Result: Working code generation

3. LLVM 21 API Changes
   - Challenge: Opaque pointers, no getPointerElementType
   - Solution: Track types separately
   - Result: Compatible with LLVM 21

4. Type Promotion
   - Challenge: Mixed int/float operations
   - Solution: Automatic type promotion in IR
   - Result: Correct float arithmetic

5. Short-Circuit Evaluation
   - Challenge: and/or need special handling
   - Solution: PHI nodes and basic blocks
   - Result: Proper short-circuit behavior
