================================================================================
USAGE
================================================================================

Build:
```bash
make quick
```

Run (Interpreter):
```bash
./sapp program.spp
```

Compile to IR:
```bash
./sapp compile program.spp
```

Compile to Native:
```bash
./scripts/demo_compiler.sh program.spp
```

Manual Compilation:
```bash
./sapp compile program.spp > program.ll
llc -filetype=obj program.ll -o program.o
clang program.o -o program
./program
```

Check Version:
```bash
./sapp --version
# Output: Sapphire v0.4.0-alpha (Milestone 3 Complete - Native Compilation)
```

================================================================================
EXAMPLE PROGRAM
================================================================================

Source (demo.spp):
```sapphire
# Sapphire Demo
let x = 100
let y = 50

print(x + y)  # 150
print(x * y)  # 5000

let pi = 3.14159
print(pi * 2)  # 6.283180

print(x + pi)  # 103.141590 (type promotion)

print(x > y)  # true
print(true and true)  # true
```

Compile and Run:
```bash
./scripts/demo_compiler.sh demo.spp
```

Output:
```
150
5000
6.283180
103.141590
true
true
```

Perfect! ✨
