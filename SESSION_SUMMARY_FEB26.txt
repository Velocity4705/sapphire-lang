================================================================================
EXAMPLE OUTPUT
================================================================================

Input (simple_test.spp):
```sapphire
let x = 10
let y = 20
let sum = x + y
let product = x * y
let difference = y - x
```

Generated IR:
```llvm
define i32 @main() {
entry:
  %x = alloca i64, align 8
  store i64 10, ptr %x, align 4
  %y = alloca i64, align 8
  store i64 20, ptr %y, align 4
  %x1 = load i64, ptr %x, align 4
  %y2 = load i64, ptr %y, align 4
  %addtmp = add i64 %x1, %y2
  %sum = alloca i64, align 8
  store i64 %addtmp, ptr %sum, align 4
  %x3 = load i64, ptr %x, align 4
  %y4 = load i64, ptr %y, align 4
  %multmp = mul i64 %x3, %y4
  %product = alloca i64, align 8
  store i64 %multmp, ptr %product, align 4
  %y5 = load i64, ptr %y, align 4
  %x6 = load i64, ptr %x, align 4
  %subtmp = sub i64 %y5, %x6
  %difference = alloca i64, align 8
  store i64 %subtmp, ptr %difference, align 4
  ret i32 0
}
```

Perfect! ✨

================================================================================
USAGE
================================================================================

Compile to IR:
```bash
./sapp compile program.spp
```

Save IR to file:
```bash
./sapp compile program.spp > program.ll
```

Compile to native:
```bash
./sapp compile program.spp > program.ll
llc program.ll -o program.o
clang program.o -o program
./program
```

Interpreter mode (still works):
```bash
./sapp program.spp
```
