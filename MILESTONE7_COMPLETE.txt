================================================================================
USAGE EXAMPLES
================================================================================

Create New Project:
```bash
spm init my-project
cd my-project
```

Build and Run:
```bash
spm build
spm run
```

Run Tests:
```bash
spm test
```

Clean Build:
```bash
spm clean
```

Generate Docs:
```bash
spm doc
```
