WHAT WAS IMPLEMENTED
================================================================================

✅ Native print() Function
   - Declares printf from C standard library
   - Generates correct format strings for each type
   - Supports int, float, bool types
   - Multiple arguments work
   - Compiles to native code

✅ Type-Aware Printing
   - int: Uses "%lld\n" format
   - float: Uses "%f\n" format
   - bool: Converts to "true" or "false" strings
   - Automatic type detection at compile time

✅ FFI (Foreign Function Interface)
   - Links with C standard library
   - External function declarations
   - Variadic function support (printf)

================================================================================
TECHNICAL IMPLEMENTATION
================================================================================

1. Printf Declaration
```cpp
llvm::FunctionType* printf_type = llvm::FunctionType::get(
    llvm::Type::getInt32Ty(*context),
    {llvm::PointerType::get(*context, 0)},  // format string
    true  // variadic
);
llvm::Function* printf_func = llvm::Function::Create(
    printf_type,
    llvm::Function::ExternalLinkage,
    "printf",
    module
);
```

2. Format String Generation
```cpp
// Integer
format_str = builder->CreateGlobalStringPtr("%lld\n");
printf_args = {format_str, value};

// Float
format_str = builder->CreateGlobalStringPtr("%f\n");
printf_args = {format_str, value};

// Boolean
true_str = builder->CreateGlobalStringPtr("true\n");
false_str = builder->CreateGlobalStringPtr("false\n");
format_str = builder->CreateSelect(value, true_str, false_str);
printf_args = {builder->CreateGlobalStringPtr("%s"), format_str};
```

3. Printf Call
```cpp
builder->CreateCall(printf_func, printf_args);
```

================================================================================
GENERATED IR EXAMPLE
================================================================================

Input:
```sapphire
print(42)
print(3.14)
print(true)
```

Generated IR:
```llvm
@int_fmt = private unnamed_addr constant [6 x i8] c"%lld\0A\00"
@float_fmt = private unnamed_addr constant [4 x i8] c"%f\0A\00"
@true_str = private unnamed_addr constant [6 x i8] c"true\0A\00"
@bool_fmt = private unnamed_addr constant [3 x i8] c"%s\00"

define i32 @main() {
entry:
  %0 = call i32 (ptr, ...) @printf(ptr @int_fmt, i64 42)
  %1 = call i32 (ptr, ...) @printf(ptr @float_fmt, double 3.140000e+00)
  %2 = call i32 (ptr, ...) @printf(ptr @bool_fmt, ptr @true_str)
  ret i32 0
}

declare i32 @printf(ptr, ...)
```

Clean, efficient, and correct! ✨
