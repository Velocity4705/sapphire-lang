================================================================================
HOW TO USE
================================================================================

Build:
  make quick

Run a program:
  ./sapp examples/hello.spp
  ./sapphire examples/simple.spp

Compile to LLVM IR:
  ./sapp compile examples/simple.spp

Run tests:
  make test-runtime

Check version:
  ./sapp --version

| Java     | ✅            | ✅          | ✅              | Runtime+GC  |
| Sapphire | ✅            | ✅          | ✅              | Runtime+GC  |

Performance:
| Language | Speed      | GC Pause | Allocation |
|----------|-----------|----------|------------|
