# Macro System

Sapphire's macro system enables compile-time metaprogramming, code generation, and DSL creation.

## Types of Macros

### Declarative Macros

Pattern-based macros that expand code at compile time.

```sapphire
macro! debug(expr) {
    print(f"[DEBUG] {stringify!(expr)} = {expr}")
}

// Usage
let x = 42
debug!(x)  // Prints: [DEBUG] x = 42
```

### Procedural Macros

Macros that operate on the AST (Abstract Syntax Tree).

```sapphire
#[proc_macro]
fn route(path: String, handler: fn) {
    register_route(path, handler)
}

#[route("/api/users")]
fn handle_users(req: Request) -> Response {
    return Response::json(get_users())
}
```

### Attribute Macros

Macros that modify items they're attached to.

```sapphire
#[derive(Serialize, Deserialize)]
type User {
    id: i64
    name: String
    email: String
}
```

## Basic Macros

### Simple Expansion

```sapphire
macro! unless(condition, body) {
    if !condition {
        body
    }
}

unless!(x < 0, {
    print("x is non-negative")
})
```

### With Repetition

```sapphire
macro! vec(elements...) {
    {
        let v = Vec::new()
        $(v.push(elements))*
        v
    }
}

let numbers = vec![1, 2, 3, 4, 5]
```

## Code Generation

### Property Generation

```sapphire
macro! property(name, type) {
    fn get_$name(self) -> $type {
        return self.$name
    }
    
    fn set_$name(self, value: $type) {
        self.$name = value
    }
}

type Person {
    name: String
    age: i64
    
    property!(name, String)
    property!(age, i64)
}
```

### Builder Pattern

```sapphire
macro! builder(struct_name, fields...) {
    type ${struct_name}Builder {
        $(field: Option<$field_type>),*
    }
    
    impl ${struct_name}Builder {
        fn new() -> Self {
            return Self { $(field: None),* }
        }
        
        $(
            fn $field(self, value: $field_type) -> Self {
                self.$field = Some(value)
                return self
            }
        )*
        
        fn build(self) -> $struct_name {
            return $struct_name {
                $(field: self.$field.unwrap()),*
            }
        }
    }
}

type Config {
    host: String
    port: i64
}

builder!(Config, host: String, port: i64)

let config = ConfigBuilder::new()
    .host("localhost")
    .port(8080)
    .build()
```

## Compile-Time Assertions

```sapphire
macro! static_assert(condition, message) {
    #[compile_time]
    if !condition {
        compile_error!(message)
    }
}

static_assert!(size_of::<i64>() == 8, "i64 must be 8 bytes")
```

## Test Generation

```sapphire
macro! test_cases(fn_name, cases...) {
    $(
        #[test]
        fn test_${fn_name}_$case_name() {
            let result = $fn_name($input)
            assert_eq!(result, $expected)
        }
    )*
}

fn add(a: i64, b: i64) -> i64 {
    return a + b
}

test_cases!(add,
    case1: (1, 2) => 3,
    case2: (5, 7) => 12,
    case3: (-1, 1) => 0
)
```

## DSL Creation

### HTML DSL

```sapphire
macro! html(content) {
    parse_html!(content)
}

fn create_page() -> Html {
    return html! {
        <div class="container">
            <h1>Welcome to Sapphire</h1>
            <p>This is generated by macros!</p>
        </div>
    }
}
```

### SQL DSL

```sapphire
macro! sql(query) {
    compile_sql!(query)
}

fn get_users() -> Vec<User> {
    return sql! {
        SELECT id, name, email
        FROM users
        WHERE age > 18
    }
}
```

## Derive Macros

### Automatic Trait Implementation

```sapphire
macro! derive_serialize(struct_name) {
    impl Serialize for $struct_name {
        fn serialize(self) -> String {
            let json = "{"
            $(json += f"\"{field_name}\": {self.$field_name},")* 
            json += "}"
            return json
        }
    }
}

type User {
    id: i64
    name: String
}

derive_serialize!(User)
```

### Common Derives

```sapphire
#[derive(Debug, Clone, PartialEq, Serialize, Deserialize)]
type Point {
    x: f64
    y: f64
}
```

## Enum Generation

```sapphire
macro! generate_enum(name, variants...) {
    type $name {
        $($variant),*
    }
    
    impl $name {
        fn to_string(self) -> String {
            match self {
                $(Self::$variant => stringify!($variant)),*
            }
        }
        
        fn from_string(s: String) -> Option<Self> {
            match s {
                $(stringify!($variant) => Some(Self::$variant)),*
                _ => None
            }
        }
    }
}

generate_enum!(Color, Red, Green, Blue, Yellow)
```

## Benchmarking

```sapphire
macro! bench(name, code) {
    fn bench_$name() {
        let start = time::now()
        for _ in 0..1000000 {
            code
        }
        let elapsed = time::now() - start
        print(f"Benchmark {name}: {elapsed}ms")
    }
}

bench!(addition, {
    let x = 1 + 2
})
```

## Macro Hygiene

Macros in Sapphire are hygienic - they don't accidentally capture variables.

```sapphire
macro! swap(a, b) {
    let temp = a
    a = b
    b = temp
}

let temp = 1
let x = 2
let y = 3
swap!(x, y)
// temp is still 1 - not affected by macro's temp variable
```

## Macro Debugging

### Expand Macros

```bash
sapp --expand-macros file.spp
```

### Show Macro Expansion

```sapphire
#[trace_macros(true)]
debug!(x)
#[trace_macros(false)]
```

## Built-in Macros

- `stringify!(expr)` - Convert expression to string
- `concat!(a, b, ...)` - Concatenate at compile time
- `file!()` - Current file name
- `line!()` - Current line number
- `column!()` - Current column number
- `compile_error!(msg)` - Emit compile error
- `compile_warning!(msg)` - Emit compile warning
- `include!(file)` - Include file contents
- `include_str!(file)` - Include file as string
- `include_bytes!(file)` - Include file as bytes

## Advanced Patterns

### Recursive Macros

```sapphire
macro! count(items...) {
    count_impl!(0, items...)
}

macro! count_impl(n, item, rest...) {
    count_impl!(n + 1, rest...)
}

macro! count_impl(n) {
    n
}
```

### Conditional Compilation

```sapphire
macro! cfg_if(condition, then_code, else_code) {
    #[cfg(condition)]
    then_code
    
    #[cfg(not(condition))]
    else_code
}

cfg_if!(target_os = "linux",
    { print("Running on Linux") },
    { print("Not running on Linux") }
)
```

## Best Practices

1. **Use macros sparingly** - Prefer functions when possible
2. **Document macro behavior** - Explain what the macro does
3. **Test macro expansions** - Verify generated code
4. **Keep macros simple** - Complex macros are hard to debug
5. **Use hygiene** - Don't capture variables accidentally
