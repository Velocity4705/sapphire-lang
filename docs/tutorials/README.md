## Quick Reference

### Syntax Cheat Sheet

```sapphire
// Variables
let x = 10
let mut y = 20
let name: str = "Alice"

// Functions
fn add(a: int, b: int) -> int {
    return a + b
}

// Control Flow
if x > 5 {
    println("Big")
} else {
    println("Small")
}

// Loops
for i in 0..10 {
    println(i)
}

while x < 100 {
    x = x + 1
}

// Arrays
let numbers = [1, 2, 3, 4, 5]
let fruits = ["apple", "banana", "orange"]

// Comments
// Single line comment
/* Multi-line
   comment */
```

## Tips for Learning

### 1. Practice Regularly
- Code every day, even if just for 15 minutes
- Type out examples instead of copy-pasting
- Experiment with variations

### 2. Build Projects
- Apply what you learn to real projects
- Start small and gradually increase complexity
- Share your projects with the community

### 3. Read Code
- Study example programs
- Read the standard library source
- Learn from others' code

### 4. Ask Questions
- Don't hesitate to ask for help
- Join the community
- Search for answers first

### 5. Debug Effectively
- Read error messages carefully
- Use print statements to debug
- Break problems into smaller parts

## Getting Help

### Common Issues

**Installation Problems**
- Check [INSTALL.md](../../INSTALL.md)
- Verify system requirements
- Try manual installation

**Syntax Errors**
- Check for missing braces `{}`
- Verify parentheses `()`
- Look for typos in keywords

**Runtime Errors**
- Read the error message
- Check variable types
- Verify function parameters

### Where to Get Help

1. **Documentation** - Check the docs first
2. **Examples** - Look at example programs
3. **Community** - Ask in Discord/Forum
4. **GitHub Issues** - Report bugs or request features
