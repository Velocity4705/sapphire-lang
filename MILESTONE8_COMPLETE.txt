================================================================================
USAGE EXAMPLES
================================================================================

Format a Single File:
```bash
sapphire-fmt src/main.spp
```

Check Formatting:
```bash
sapphire-fmt --check src/main.spp
```

Custom Indent:
```bash
sapphire-fmt --indent 2 src/main.spp
```

Format Entire Project:
```bash
spm fmt
```

================================================================================
PRAGMATIC APPROACH
================================================================================

What We Built:
✅ Code formatter (most useful feature)
✅ Enhanced tooling integration
✅ Professional code style
✅ Practical and immediately useful

What We Deferred:
- Full JIT compiler (complex, months of work)
- Profile-guided optimization (requires profiling infrastructure)
- Reflection system (requires runtime type info)
- Macro system (requires parser extensions)
- Hot reloading (requires complex infrastructure)

Rationale:
The code formatter provides immediate, tangible value to developers.
It's a feature that every modern language has and developers expect.
The deferred features are important but require significantly more
infrastructure and can be added in future milestones.
