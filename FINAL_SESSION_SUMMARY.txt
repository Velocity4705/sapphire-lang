WHAT WORKS RIGHT NOW
================================================================================

1. Interpreter Mode
   ./sapp program.spp
   → Runs Sapphire programs
   → Variables, arithmetic, control flow

2. Compiler Mode (Show IR)
   ./sapp compile program.spp
   → Generates optimized LLVM IR
   → Shows intermediate representation

3. Native Compilation (External Tools)
   ./sapp compile program.spp > program.ll
   llc program.ll -o program.o
   clang program.o -o program
   ./program
   → Native executable!
   → Full performance!



```bash

```

Step 3: Run
```bash
./test
```
